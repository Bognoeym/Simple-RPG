# Simple RPG
C++의 다양한 개념 응용과 많은 소스, 헤더 파일에 익숙해지기 위해 제작해보았다.
- C/C++, Visual Studio  
- 콘솔창에서 즐기는 가위바위보 RPG 게임
- 플레이 영상 :: https://youtu.be/xdj2MFk9XgE

## 조작법
- **w**: 위  
- **s**: 아래  
- **Enter**: 선택
